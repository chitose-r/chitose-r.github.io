Pytorch(8)-单层神经网络

发表于2023-12-19|更新于2024-01-04|Pytorch
|字数总计:670|阅读时长:3分钟|阅读量:

1.单层线性回归

  • tensor实现神经网络的正向传播
    • 特征张量x最好设置为float32规避bug
    • 标签张量z最好设置为二维，当有bug再view
      • pytorch中的nn.层中最后输出的是二维结果
    • 规避精度问题：torch.allclose()
tensor([-0.2000, -0.0500, -0.0500,  0.1000])
True
  • torch.nn.Linear实现神经网络的正向传播
    • torch.nn.Linear(上一层神经元的个数,这一层神经元的个数)
    • 所有nn.module的层都会实例化带梯度的w和b
    • zhat计算结果为二维
tensor([[0.6730],
        [1.1048],
        [0.2473],
        [0.6792]], grad_fn=<AddmmBackward0>)
Parameter containing:
tensor([[ 0.4318, -0.4256]], requires_grad=True)
Parameter containing:
tensor([0.6730], requires_grad=True)

2.单层逻辑回归

  • sigmoid函数：对数几率回归

    • 设置阈值为0.5(默认),可得预测为0/1
    • 化回归为分类

  • tensor实现二分类神经网络的正向传播
